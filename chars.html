立绘 (Character) 控制指南

精确控制角色的位置、层级、大小和动画效果

← 返回主页

1. 基本语法结构

立绘指令采用方括号包裹的形式：[修饰词 角色ID]。如果需要同时控制多个立绘，可以使用逗号分隔。

// 显示 ID 为 lh01 的立绘，位置居中 chars: "[中 lh01]" // 同时显示两个立绘 chars: "[左 lh01],[右 lh02]"

2. 水平位置控制

关键词屏幕百分比 (Left)说明
左左15%最左侧
左25%偏左侧
左右35%微偏左
中50%屏幕正中央（默认值）
右左65%微偏右
右75%偏右侧
右右85%最右侧

3. 垂直位置与层级

通过关键词控制立绘的垂直偏移（Bottom）和堆叠顺序（zIndex）。

  • 垂直： 下 (-25%), 中下 (-50%), 下下 (-65%)
  • 层级： 前 (zIndex 11), 后 (zIndex 9)

4. 缩放比例

使用带有 % 符号的数字调整大小。正值放大，负值缩小。

chars: "[中 15% lh01]" // 放大到 115% chars: "[中 -20% lh01]" // 缩小到 80%

5. 动画控制

提示： 默认情况下，立绘变化会带有 0.5 秒的平滑过渡动画。若出现卡顿，可使用 瞬 指令禁用动画。
chars: "[瞬 左 lh01]" // 瞬间移动，无动画

6. 组合指令示例

您可以将多个修饰词组合在一个指令中，顺序不影响解析结果。

// 完整组合：瞬移 + 位置 + 缩放 + 层级 chars: "[瞬 中 下下 20% 前 lh01]"

7. 清除立绘

chars: "[消失 lh01]"
