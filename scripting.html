基础脚本语法与指令规范

掌握 sceneData 结构与 Command 字段的核心用法

← 返回主页

1. 场景数据结构 (sceneData)

每个场景文件包含一个 `sceneData` 对象，定义了背景、音频和故事脚本。

const sceneData = {
    background: BG_CONFIG_SUB,
    bgm: BGM_CONFIG_SUB,
    story: [
        {
            text: "对话文本内容",
            speaker: "说话者姓名",
            background: "bg_001_00_00",
            command: "[pov 主角]"
        }
    ]
};

2. Command 字段使用规范

黄金法则： 一行只做一件事。要么显示文本，要么执行命令，不要混用。

[错误] 错误用法：命令与文本混用

{
    text: "这段文字会被跳过",
    command: "[fadeout time=1000]"
}

[正确] 正确用法：独立成行

{
    text: "即将开始转场...",
    speaker: "旁白"
},
{
    text: "",
    command: "[fadeout time=1000]"
}

多指令并行支持

可以在单个 command 属性中使用英文逗号 , 分隔多个指令，引擎会按顺序依次执行：

{
    text: "",
    command: "[normal],[pov 主角],[lock]" // 切换模式、显示POV、锁定点击
}

这种方式可以大幅减少代码行数，提高编写效率。多个指令会在同一帧内按顺序执行，适合不需要间隔的连续操作。

command 与 action 共存

可以在同一个剧情对象中同时使用 command 和 action 属性，引擎会先执行 command 指定的指令，再执行 action 指定的动作：

{
    text: "",
    command: "[msgoff]", // 先隐藏文本框
    action: { // 然后显示选项菜单
        type: "choice",
        choices: [
            { text: "选项1", target: "scene2.html" },
            { text: "选项2", target: "scene3.html" }
        ]
    }
}

这种写法简化了代码结构，常用于在显示选项前执行一些预处理指令（如隐藏文本框、切换模式、锁定点击等）。

3. Action 动作对象

action 属性用于执行特殊动作，如场景跳转、显示选项等。

直接跳转到下一个页面

{
    text: "",
    action: {
        type: "nextScene",
        target: "scene2.html" // 输入需要跳转的网页名称
    }
}

通过选项跳转到对应页面

{
    text: "",
    action: {
        type: "choice",
        choices: [
            { text: "前往场景1", target: "scene1.html" },
            { text: "返回主菜单", target: "../index.html" }
        ]
    }
}

保持选项页面整洁

在显示选项前，可以配合 command 属性隐藏文本框，使选项界面更加整洁：

{
    text: "",
    command: "[msgoff]", // 隐藏文本框
    action: {
        type: "choice",
        choices: [
            { text: "前往场景1", target: "scene1.html" },
            { text: "返回主菜单", target: "../index.html" }
        ]
    }
}

4. 常用指令速查

指令功能说明
[pov 角色名]在右上角显示当前叙事视角
[pov stop]隐藏视角指示器
[wait]暂停剧情，等待用户点击
[fadeout time=1000]屏幕淡出效果
[novel]开启全屏小说模式
[normal]关闭全屏小说模式
[lock]、[阻止]锁定鼠标点击，屏蔽非快进模式的点击（按住 Ctrl 仍可快进）
[free]、[解锁]解除鼠标点击锁定，恢复正常点击交互

5. 文本格式标签

  • \n：换行符，用于在同一段落内换行。
  • [s]：分段符，点击鼠标后显示下一段文本。
  • [汉字,拼音]：注音标注，为文字添加上方注音（类似日语假名）。

注音功能示例

使用 [汉字,拼音] 语法可以为文本添加注音标注：

text: "这是一段[测试,ce shi]文本"

显示效果："测试"二字上方会显示拼音 "ce shi"。

多语言支持

// 中文拼音标注
text: "这是[中文,zhong wen]注音"

// 日语假名标注
text: "这是[日语,にほんご]标注"

// 混合使用
text: "[你好,ni hao]，[世界,shi jie]！"

注意事项：

  • 注音功能自动适配常规模式和全屏小说模式
  • 适用于所有HTML支持的语言
  • 注音内容会以内联方式显示在对应文字上方
text: "第一段文字[s]第二段文字\n第三行文字"
